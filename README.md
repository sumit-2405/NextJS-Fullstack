A full-stack private video sharing application built with Next.js (App Router) that allows users to securely upload, manage, and watch their own private videos.
Videos are accessible only after authentication and are delivered efficiently using ImageKit.

## Getting Started

First, install all the dependencies:

```bash
npm install

```

Then, run the development server

```bash
npm run dev

```


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
